# neotest-kotlin

This is an adapter for the neotest project. This gives you the ability to see and run your tests within neovim.

```lua
-- lazy.nvim setup
{
  "nvim-neotest/neotest",
  dependencies = {
    -- ...
    "codymikol/neotest-kotlin"
  },
  config = function()
    require("neotest").setup({
      adapters = {
        require("neotest-kotlin")
      }
    })
  end
}
```
